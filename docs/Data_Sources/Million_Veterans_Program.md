# Million Veterans Program


The Million Veterans Program (MVP) is a longitudinal health cohort study organized by the US Department of Veterans Affairs, which enrolled its one-millionth participant in 2023[@verma2024diversity].
 
MVP participants are significantly more ethnically diverse than participants in other studies of a similar scale.


The MVP investigators have run a large number of GWAS using genetic and phenotypic data collected from participants, and have publicly released summary statistics from these studies.
