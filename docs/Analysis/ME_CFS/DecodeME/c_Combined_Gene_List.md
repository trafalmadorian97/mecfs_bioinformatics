# Gene List

This file aggregates [DecodeME](../../../Data_Sources/DecodeME.md) candidate genes generated by [GWASLAB](a_DecodeME_Lead_Variants.md) and [MAGMA](g_MAGMA_DecodeME_Analysis.md).  Each gene is accompanied by annotations retrieved using [gget](https://github.com/pachterlab/gget) from bioinformatic databases.
